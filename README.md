# Home Utility Ledger

自建 Docker 家庭水电燃账单系统。

## 当前能力

- 管理员登录保护，未登录访问会跳转到登录页。
- 前台看板：本月合计、电费、水费、燃气费、最近账单、采集状态。
- 统计分析：日、月、季、年维度接口与页面。
- 后台配置：国网浙江、杭州市水务、杭州天然气接入。
- 国网浙江：导入已登录 95598 页面生成的登录状态 JSON 后自动采集月账单和近 7 日日用电。
- 杭州市水务：导入杭水 e 家小程序 HAR 后自动识别会话并采集账单。
- 杭州天然气：导入公众号账单页 HAR 后自动识别会话并采集账单。
- SQLite 持久化到 `/data`。

## 本地运行

```bash
docker compose up -d --build
```

访问：

```text
http://服务器IP:3000
```

默认账号：

```text
admin / 你在 compose 中设置的 ADMIN_PASSWORD
```

部署前必须把 `ADMIN_PASSWORD` 改成自己的强密码。

## Portainer Stack

如果你已经发布了 GHCR 镜像，把 `docker-compose.portainer.yml` 里的镜像名改成实际地址后粘贴到 Portainer。

持久化目录：

```text
/home/docker/home-utility-ledger
```

## 数据接入说明

### 国网浙江电力

在已经登录的国网页面导出登录状态 JSON，粘贴到后台“国网浙江电力 -> 导入登录状态”。

### 杭州市水务

抓取“杭水 e 家”小程序账单或水表页面 HAR，粘贴到后台“杭州市水务 -> 导入抓包文件”。

### 杭州天然气

抓取杭州天然气公众号账单查询页 HAR，粘贴到后台“杭州天然气 -> 导入抓包文件”。

## 说明

第三方会话可能过期。过期时后台测试连接会明确提示失败，需要重新导入登录状态或抓包文件。
